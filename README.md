#  AR Ruler App

An Augmented Reality (AR) measurement application built with Swift and ARKit.

This project was developed as part of the iOS Development Bootcamp by Angela Yu.  
The app allows users to measure real-world distances using the device camera and AR technology.

---

##  Features

-  Real-time camera-based measurement
-  Detecting horizontal surfaces
-  Placing 3D points in AR space
-  Calculating distance between two points
-  Reset measurement functionality
-  Accurate distance calculation using vector math

---

##  Technologies Used

- Swift
- UIKit
- ARKit
- SceneKit
- Xcode
- 3D Vector Mathematics

---

##  How It Works

1. The app uses ARKit to detect horizontal planes.
2. The user taps on the screen to place 3D points in the real world.
3. The app calculates the distance between two selected points.
4. The measured distance is displayed on the screen.
